Cấu hình để đồng bộ dữ liệu phim với tất cả người dùng. Web sẽ đọc file data.json từ GitHub thay vì localStorage. Admin bấm "Đẩy lên GitHub" để cập nhật bản chung.

Mặc định: web = khang2026, admin = khangadmin2026. Để đồng bộ GitHub, tạo token tại: Settings → Developer Settings → Personal Access Tokens → Generate new token (classic), tích quyền repo.
